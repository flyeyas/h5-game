# 游戏详情页Header全宽设计报告

## 🎯 修改目标

将游戏详情页的header区域样式设置为全宽，让header背景延伸到浏览器窗口的整个宽度。

## ✅ 修改内容

### CSS样式调整
**文件**: `templates/games/game_detail.html`
**位置**: 第64-72行

#### 修改前
```css
/* 游戏头部区域 - 严格按照原型 */
.game-header {
    background: linear-gradient(135deg, #6a11cb 0%, #2575fc 100%);
    color: white;
    padding: 40px 0;
    margin-bottom: 30px;
}
```

#### 修改后
```css
/* 游戏头部区域 - 全宽设计 */
.game-header {
    background: linear-gradient(135deg, #6a11cb 0%, #2575fc 100%);
    color: white;
    padding: 40px 0;
    margin-bottom: 30px;
    width: 100vw;
    margin-left: calc(-50vw + 50%);
    position: relative;
}
```

## 🎨 技术实现

### 全宽CSS技术
**核心CSS属性**:

1. **`width: 100vw`**
   - 设置宽度为视口宽度的100%
   - 确保header覆盖整个浏览器窗口宽度

2. **`margin-left: calc(-50vw + 50%)`**
   - 使用CSS calc()函数计算负边距
   - 将元素向左移动，使其从视口左边缘开始
   - 公式解释：-50vw（向左移动视口宽度的一半）+ 50%（向右移动元素宽度的一半）

3. **`position: relative`**
   - 确保元素定位正确
   - 为可能的子元素绝对定位提供参考

## 🔍 实现细节

### CSS计算解释
```css
width: 100vw;                    /* 宽度 = 视口宽度 */
margin-left: calc(-50vw + 50%);  /* 左边距计算 */
```

**计算过程**:
1. 元素原本在容器中居中
2. `width: 100vw` 使宽度扩展到视口宽度
3. `margin-left: calc(-50vw + 50%)` 将元素左边缘移动到视口左边缘
4. 内部的`.container`仍然保持居中布局

### 布局影响
- **不影响其他元素** - 只改变header的显示方式
- **保持内容居中** - 内部container布局不变
