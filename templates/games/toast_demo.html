Toast提示演示

这个页面演示了不同类型的Toast提示，以及如何使用它们。

基本提示类型

这些是基本的提示类型，用于不同场景下的用户反馈。

// 成功提示
toast.success('操作成功完成！');

// 错误提示
toast.error('发生了错误，请稍后再试。');

// 信息提示
toast.info('这是一条信息提示。');

// 警告提示
toast.warning('注意：此操作不可撤销。');

提示位置

Toast提示可以在屏幕的不同位置显示。

// 显示在右上角
toast.info('右上角提示', { position: 'top-right' });

// 显示在顶部居中
toast.info('顶部居中提示', { position: 'top-center' });

// 显示在左上角
toast.info('左上角提示', { position: 'top-left' });

// 其他位置: 'bottom-right', 'bottom-center', 'bottom-left'

提示持续时间

可以自定义Toast提示的显示时间。

// 短暂提示（1.5秒）
toast.info('短暂提示', { duration: 1500 });

// 标准提示（3秒，默认值）
toast.info('标准提示');

// 较长提示（5秒）
toast.info('较长提示', { duration: 5000 });

// 持久提示（不自动关闭，需要用户手动点击关闭按钮）
toast.info('持久提示', { duration: 0 });

常见使用场景

以下是一些在AJAX请求中使用Toast提示的常见场景。

// 使用API请求函数进行AJAX请求
apiRequest({
    url: '/ajax/example-endpoint/',
    method: 'POST',
    data: { key: 'value' },
    showLoading: true,         // 显示加载提示
    loadingMessage: '正在提交...',  // 自定义加载提示文字
    onSuccess: function(data) {
        // 处理成功响应，会自动显示data.message作为成功提示
    },
    onError: function(error) {
        // 处理错误，会自动显示error.message作为错误提示
    }
});

全局配置

你可以更改Toast提示的全局配置，影响所有后续的提示。

// 设置全局配置
toast.configure({
    position: 'top-right',     // 默认位置
    duration: 3000,            // 默认持续时间
    showProgress: true,        // 显示进度条
    pauseOnHover: true,        // 鼠标悬停暂停倒计时
    darkMode: false            // 暗色主题
});
